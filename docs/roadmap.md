# Roadmap: StarBridgeTutoring

This roadmap outlines the evolution of StarBridgeTutoring into a fully-fledged interactive EdTech platform.

---

## MVP (Minimum Viable Product)
**Focus:** Core Real-time Classroom and Essential Scaffolding

- [x] Basic React/Vite scaffolding.
- [x] Integrate Tailwind CSS with YELLOW primary theme.
- [x] Implement Supabase Authentication (Teacher login & Student Join-by-Code).
- [x] Set up PostgreSQL Tables via Supabase (`users`, `classes`, `lessons`).
- [x] Teacher Dashboard: Create simple lessons and upload static slides (Images/PDFs via Supabase Storage).
- [x] Real-time Synchronization: Connect Teacher screen to Student screens using Supabase Realtime (WebSocket).

---

## Phase 2
**Focus:** Interactivity and The "Lumio Killer" Features

- [x] Interactive Quizzes: Teachers can broadcast multiple-choice questions.
- [x] Shout-It-Out Board: A collaborative real-time board where students post sticky notes.
- [x] Polls & Pulse Checks.
- [x] Enhanced Mobile Optimization: ensuring touch-friendly interfaces for the student classroom.
- [x] Basic Analytics: Track student attendance and participation rates per lesson.

---

## Phase 3
**Focus:** Specialized Modules and Ecosystem Growth

- [x] **Islamic Classes Module:** Dedicated components for Quran reading, Fiqh, and Hadith studies.
- [x] Gamification & Badging system for students.
- [x] Advanced Teacher Insights: Downloadable CSV reports on student performance.
- [x] Whiteboarding: Allow teachers to draw on slides in real time (using tldraw).
- [x] Offline Support / PWA functionalities for students in extreme low-bandwidth areas (Service Workers).
